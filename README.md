## Overview
A **neo-brutalist** single-page portfolio with a card-based UI.  
It includes 3 main sections:

- **Home** – Personal info, summary, education, achievements  
- **About** – Experience and skills  
- **Projects** – Showcase of AI & ML projects  

Tabs at the top switch between sections using simple JavaScript.

---

## Features
- Bold borders, rotated elements, and heavy shadows  
- Interactive tab navigation  
- Responsive and mobile-friendly design  
- Stylized cards for skills, experience, and projects  
- Minimal JavaScript

---

## Tech Stack
- **HTML** – Structure  
- **CSS** – Styling & layout  
- **JavaScript** – Section switching
